# 🚀 Youtube Video Analyzer

An intelligent **Agentic AI-powered web application** that analyzes content from user-provided links and generates structured insights such as **timestamp breakdowns, summaries, and contextual understanding** using a high-performance LLM.

---

## 🌟 Features

* 🔗 **Link-Based Analysis**
  Paste any supported content link (e.g., videos/articles) and get instant insights.

* ⏱️ **Timestamp-Wise Breakdown**
  Automatically segments content into meaningful time-based chunks.

* 🧠 **AI-Powered Summarization**
  Generates concise and context-aware summaries.

* 📊 **Structured Output**
  Clean and readable results for better understanding.

* ⚡ **High-Speed Processing**
  Optimized using Groq LLM for fast inference.

---

## 🧠 How It Works

1. User submits a content link
2. System extracts and preprocesses content
3. Agentic AI pipeline processes the data
4. LLM generates:

   * Timestamp summaries
   * Key insights
   * Contextual analysis
5. Results are displayed in a structured format

---

## 🏗️ Tech Stack

* **Frontend:** Streamlit
* **Backend:** Python
* **LLM:** Groq (LLM API)
* **AI Framework:** Agentic Workflow (Custom Prompt Engineering)
* **Tools:**

  * Content parsing utilities
  * API integrations


---

## ▶️ Usage

```bash
streamlit run app.py
```

1. Open the app in your browser
2. Paste a content link
3. Click analyze
4. View AI-generated insights
